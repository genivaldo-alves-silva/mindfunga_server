# 1. Cadastro, Configuração e Primeiro acesso

## 1.1 Cadastro
- Para cadastrar teu usuário no servidor, contacte os administradores.

## 1.2 Configuração da VPN (faça antes do primeiro acesso)
- Abra os guias da UFSC para configurar a VPN no teu sistema (exemplo para abrir no host):
  - Ubuntu: [vpnUFSC/ubuntu](https://servicosti.sistemas.ufsc.br/publico/faq.xhtml?faq=5459)
  - macOS: `"$BROWSER" https://servicosti.sistemas.ufsc.br/publico/faq.xhtml?faq=5039`
  - Windows 11: `"$BROWSER" https://servicosti.sistemas.ufsc.br/publico/faq.xhtml?faq=6105`

## 1.3 Primeiro acesso ao servidor (SSH)
- Depois de ter teu usuário, senha e VPN ativa, substitua `SEU_USUARIO` pelo teu usuário e conecte:
```bash
# Conectar ao servidor (substitua SEU_USUARIO)
ssh SEU_USUARIO@server.ip.address
# Quando solicitado, digite sua senha
```
- Usuários Windows já têm instalado o OpenSSH, que habilita o uso do ssh acima. Caso não funcione, tente seguir a instalação do OpenSSH Client no link abaixo:
`"$BROWSER" https://learn.microsoft.com/pt-br/windows-server/administration/openssh/openssh_install_firstuse`
- Usuários Linux sabem o que fazer.

## 1.4 Comandos básicos para testes (no terminal)
- Comandos úteis para verificar localização, listar e criar pastas:

```bash
# a cada comando aperte ENTER
# mostrar caminho do diretório atual
pwd

# listar arquivos e pastas
ls

# criar uma pasta chamada "filogenia"
mkdir filogenia

# verificar novamente
ls

# entrar na pasta criada
cd filogenia
```
