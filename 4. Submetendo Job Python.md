# Submetendo Job Python

## Transferindo os exemplos

```bash
# no terminal do teu notebook
scp -r /home/user/Downloads/**{testhard.py,testhard.sub}** SEU_USUARIO@server.ip.address:/home/SEU_USUARIO/filogenia/

# digite a senha de acesso ao servidor
```

## No terminal do servidor:
```bash
python3 -m venv env1

source env1/bin/activate

pip install psutil

#Verifique se os arquivos .sub e .py foram transferidos
ls

condor_submit testhard.sub

# verifique o status do job
condor_q

# use watch condor_q para acompanhar em tempo real a evolução do job, em hold, idle, run, etc
watch condor_q

# explore os arquivos criados de erro e output
cat test-h.err

cat test-h.out
```

## Veja abaixo o arquivo de submissão de exemplo (testhard.sub). O executável está inclusive já executando de dentro de um ambiente (env1).

Arquivo testhard.sub, disponível em /Python job.
```sub
#testhard.sub -- RAM consuming job

executable              = /home/user/env1/bin/python3
arguments               = /home/user/test-hard.py

log                     = test-h.log
output                  = test-h.out
error                   = test-h.err

should_transfer_files   = NO

request_cpus            = 2
request_memory          = 30G
request_disk            = 5G

queue
```
